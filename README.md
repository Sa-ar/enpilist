## Run Locally

Clone the project

```bash
  git clone https://github.com/Sa-ar/enpilist.git
```

Go to the project directory

```bash
  cd elementor-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm dev
```
